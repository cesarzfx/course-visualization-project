## Project Requirements Coverage

| Requirement | Implementation |
|------------|----------------|
| 9 ACC Program Cards | All nine undergraduate programs are displayed in a responsive card-based layout |
| Required Courses | Each program includes a modal listing the required CIT, CGT, or CNIT courses |
| Career Examples | Each program includes a separate modal with potential career paths |
| Interactive Quiz | A 10-question accordion quiz helps users explore their interests |
| Responsiveness | Bootstrap grid and media queries support mobile, tablet, and desktop layouts |
| SCSS | Styling is written in SCSS and compiled into CSS |
| Hamburger Menu | Bootstrap responsive navbar collapses into a hamburger menu on smaller screens |
| Favicon | Custom favicon included and linked in the HTML head |
| Accessibility | Semantic tags, alt text, ARIA labels, and accessible Bootstrap components are used |
